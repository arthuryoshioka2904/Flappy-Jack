# Flappy Jack 🐉

A fantasy pixel-art Flappy Bird style game made with Java and Greenfoot.

## Features
- Animated dragon sprites
- Pixel art scrolling background
- Sound effects and music
- Score and High Score system
- Game Over screen
- Restart system
- Fantasy themed visuals

## Controls
- SPACE → Fly
- R → Restart after Game Over

## Technologies
- Java
- Greenfoot
